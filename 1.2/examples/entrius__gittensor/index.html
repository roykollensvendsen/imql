task:
  summary: Miners register a GitHub fine-grained personal access token (PAT) with validators and contribute
    merged pull requests (and discovered/solved issues) to whitelisted open-source repositories. Validators
    authenticate GitHub account ownership via the PAT (or the das-github-mirror service), fetch the miner's
    PRs over a rolling 35-day window, and score each merged PR by an AST/tree-sitter token-based base
    score scaled by code density, then multiplied by repo weight, linked-issue, label, open-PR-spam, time-decay,
    credibility, and review-quality multipliers. Per-miner scores are summed, normalized to sum to 1,
    and blended into four hardcoded emission pools.
  submission_format:
  - source_code

    + leaf tokens). Density-scaled SOURCE token score feeds the base score: initial_base_score = MERGED_PR_BASE_SCORE(25)
    * source_density when source token score >= MIN_TOKEN_SCORE_FOR_BASE_SCORE(5), else 0 (mirror/scoring.py
