!!! abstract "When to use this"
    You're reverse-engineering an **existing** subnet from its source code into a faithful, evidence-backed
    instance. This is the descriptive path — the discipline that makes the result trustworthy. To *read* a
    subnet that's already in the corpus, you don't need this; see
    [read an existing subnet](read-a-subnet.md) instead.
